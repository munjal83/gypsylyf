# gypsylyf
An Express based web application that lists some of the most beautiful national parks in the world. 

Technologies used for this project include:

    HTML5
    CSS3
    JavaScript
    Bootstrap
    DOM Manipulation
    jQuery
    Unix(Command Line) Commands
    NodeJS
    NPM
    ExpressJS
    REST
    MongoDB
    Database Associations
    Authentication
    PassportJS
    Authorization
